1) The implementation code for the project can be found in the
CribAgin_Code directory.

2) CribAgin_App directory contains the functional application along
with the required dependencies. Availability of JRE is a prerequisite
for running the application.

3) The test files for the application can also be found in this directory.
This directory also contains run time stats and snapshots of the 
application packed in a separate directory CribAgin_Output.
